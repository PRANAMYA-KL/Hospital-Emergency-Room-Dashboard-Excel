# Hospital-Emergency-Room-Dashboard-Excel
This project presents an interactive Hospital Emergency Room Dashboard developed in Microsoft Excel using Power Query, Power Pivot, DAX, Pivot Tables, Pivot Charts, and Slicers. It provides insights into patient demographics, operational efficiency, wait times, department performance, admissions, and patient satisfaction.

Hospital Emergency Room Dashboard
Project Overview

This project presents an interactive Hospital Emergency Room Dashboard developed in Microsoft Excel using Power Query, Power Pivot, DAX, Pivot Tables, Pivot Charts, and Slicers. It provides insights into patient demographics, operational efficiency, wait times, department performance, admissions, and patient satisfaction.
